PortSwigger Web Security Certification

Expedición: 5 de diciembre de 2025, 12:00

Organización: PortSwigger

Estado: Activa

Curso: Web Security Academy

Descripción:

Certificación oficial otorgada a David Cigaran Flores por demostrar competencia en pruebas de seguridad web, incluyendo identificación y explotación de vulnerabilidades en aplicaciones web, siguiendo los estándares de PortSwigger.

Criterios de Evaluación:

  • Detección y explotación de vulnerabilidades XSS, SQLi y CSRF.
  • Comprensión de técnicas de autenticación y control de sesiones.
  • Uso correcto de Burp Suite para pruebas de seguridad web.
  • Implementación de metodologías de seguridad web y pruebas de penetración.
  • Documentación precisa de hallazgos y mitigaciones sugeridas.

Habilidades Validadas:

Burp Suite XSS SQL Injection CSRF Authentication Testing Session Management Recon Web Security Testing

Metodologías Aplicadas:

  • Pruebas de inyección y validación de entradas
  • Análisis de vulnerabilidades en aplicaciones web
  • Evaluación de autenticación y control de sesiones
  • Uso de Burp Suite para pruebas automatizadas y manuales
  • Reporte de hallazgos y recomendaciones de mitigación
